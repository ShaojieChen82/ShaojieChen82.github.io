Distributed Energy · CHP

E8kW Micro-CHP System

Problem

Develop a residential micro combined heat and power system that can generate electricity while recovering engine heat for domestic hot water and space heating support.

My Role

Supported system architecture, controls definition, test planning, supplier communication, electrical/thermal validation, and integration discussions.

Technical Scope

  • Natural-gas engine-generator system architecture
  • PM generator, rectifier, DC bus, and inverter interface
  • Thermal recovery into hot-water tank operating strategy
  • SOC- and load-based control logic concepts
  • PLC / Modbus communication and register mapping

Evidence to Add

Power Electronics · BESS · Test

Hybrid Inverter / BESS Validation

Problem

Evaluate how a hybrid inverter behaves under bypass, backup, battery, generator, and load-transfer conditions so system architecture decisions can be made with measured evidence.

My Role

Planned and executed bench tests, instrumented voltage/current measurements, analyzed transfer behavior and ripple, and translated test results into architecture recommendations.

Technical Scope

  • Bypass current behavior and load-path validation
  • ATS transfer timing and backup behavior
  • Efficiency and power-path checks
  • DC battery charging and ripple analysis
  • Dewesoft-based instrumentation and signal review

Evidence to Add

  • Test setup photo
  • Instrument connection diagram
  • ATS timing plot
  • Efficiency curve
  • DC ripple waveform

Vehicle Mechatronics · Embedded Controls

C7 Corvette Active Aero System

Problem

Create an active aero system for a C7 Corvette that can adjust rear wing and front aero position based on driver demand and vehicle state.

My Role

Designed the control architecture, actuator logic, CAN signal integration, web/manual controls, and test approach for a personal engineering project.

Technical Scope

  • ESP32 controller with MCP2515 CAN interface
  • Rear actuator control with position feedback
  • Front servo control and linked aero modes
  • Brake, throttle, and vehicle-state based logic
  • Manual override, watchdog, filtering, and safety behavior

Evidence to Add

  • Wiring diagram
  • CAN signal table
