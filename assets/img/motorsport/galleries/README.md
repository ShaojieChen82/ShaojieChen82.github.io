# Motorsport Project Galleries

Each motorsport project gallery is a slideshow. Images rotate automatically every 5 seconds and can be clicked to enlarge.

Use numbered filenames in each folder:

```text
01.jpg or 01.png
02.jpg or 02.png
03.jpg or 03.png
...
12.jpg or 12.png
```

The site checks `.jpg`, `.png`, then `.jpeg` for each number.

## Folders

```text
assets/img/motorsport/galleries/c7-aero/
assets/img/motorsport/galleries/can-controls/
assets/img/motorsport/galleries/track-data/
assets/img/motorsport/galleries/sensors/
```

## Recommended content

- `c7-aero` — full car, wing, actuator, linkage, installation, track photos.
- `can-controls` — ESP32, MCP2515, wiring, dashboard, CAN screenshots, bench testing.
- `track-data` — PDR, Pi Toolbox, suspension travel, track maps, before/after comparisons.
- `sensors` — suspension sensors, tire thermal images, wheel-well views, instrumentation concepts.

Keep images public-safe and avoid confidential documents or unapproved internal screenshots.
